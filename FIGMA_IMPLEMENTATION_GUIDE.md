## 🎨 从 Figma 导出资源

### 方法1：使用 Figma 导出功能

1. **选择元素**
   - 在 Figma 中点击要导出的元素
   - 右侧面板会显示"Export"选项

2. **设置导出参数**
   - 格式：PNG（透明背景）或 SVG（矢量）
   - 尺寸：@1x, @2x, @3x
   - 命名：使用有意义的名称

3. **批量导出**
   - 选择多个元素
   - 点击"Export"
   - 选择导出位置

### 方法2：使用 Figma 插件

推荐插件：
- **Figma to Code** - 自动生成 HTML/CSS
- **Anima** - 导出响应式代码
- **Zeplin** - 设计交接工具

### 方法3：手动截图（临时方案）

1. 放大到 200% 或 300%
2. 使用截图工具截取
3. 使用 Photoshop 或在线工具去背景

## 💻 实现步骤

### 1. 分析设计稿

- 测量元素尺寸
- 记录颜色值
- 了解布局结构
- 标注间距和对齐

### 2. 导出资源

- 按照上述清单导出所有图片
- 保存到 `static/images/` 对应目录
- 确保命名规范

### 3. 编写 HTML 结构

参考设计稿的层级结构：
```html
<div class="poker-table-container">
  <div class="poker-table">
    <!-- 玩家座位 -->
    <div class="seat seat-0">...</div>
    <div class="seat seat-1">...</div>
    <!-- ... -->
    
    <!-- 中央区域 -->
    <div class="table-center">
      <div class="community-cards">...</div>
      <div class="pot">...</div>
    </div>
  </div>
  
  <!-- 行动按钮 -->
  <div class="action-buttons">...</div>
</div>
```

### 4. 编写 CSS 样式

- 使用 Figma 中的颜色值
- 复制字体、间距、圆角等属性
- 实现响应式布局

### 5. 添加交互

- 按钮点击效果
- 悬停动画
- 状态切换

## 💡 提示

1. **使用 Figma 的开发模式**
   - 点击右上角的"Dev Mode"
   - 可以直接复制 CSS 代码
   - 查看精确的尺寸和间距

2. **使用 Figma 插件**
   - "Figma to Code" 可以自动生成代码
   - "Anima" 可以导出响应式代码

3. **保持设计一致性**
   - 严格按照设计稿实现
   - 不要随意修改颜色和尺寸
   - 保持间距和对齐

4. **优先实现核心功能**
   - 先做游戏牌桌
   - 再做游戏大厅
   - 最后做辅助页面
