## ✅ 正确示例（目标效果）

### 关键元素

1. **牌桌本体**
   - 椭圆形轮廓清晰可见
   - 深色毛毡材质
   - 有边缘和阴影
   - 占据屏幕中央 60-70% 区域

2. **座位位置**
   - 紧贴牌桌边缘
   - 围绕椭圆均匀分布
   - 每个座位朝向中心
   - 座位之间有合理间距

3. **下注筹码**
   - 放在座位和中央之间
   - 不是放在座位上
   - 形成从座位到中央的路径

4. **中央区域**
   - 公共牌横向排列
   - 底池筹码堆在中央
   - 所有元素在牌桌内部

## 📏 座位坐标计算

### 椭圆方程

```
x = centerX + radiusX * cos(angle)
y = centerY + radiusY * sin(angle)
```

### 6人桌座位角度

```javascript
const seats = 6;
const angles = [];
for (let i = 0; i < seats; i++) {
  // 从底部中央开始，逆时针分布
  angles[i] = (Math.PI / 2) + (i * 2 * Math.PI / seats);
}

// 座位0（玩家）：底部中央
// 座位1：左下
// 座位2：左上
// 座位3：右上
// 座位4：右下
// 座位5：右侧
```

## 🎨 视觉层次

### Z-index 层级

```
1. 背景层 (z-index: 0)
   - 房间背景
   - 环境装饰

2. 牌桌层 (z-index: 10)
   - 椭圆形牌桌
   - 牌桌边框

3. 座位层 (z-index: 20)
   - 玩家座位卡片
   - 头像、昵称、筹码

4. 筹码层 (z-index: 30)
   - 下注筹码
   - 底池筹码

5. 卡牌层 (z-index: 40)
   - 公共牌
   - 玩家手牌

6. UI 层 (z-index: 50)
   - 行动按钮
   - 下注滑块
   - 提示信息
```

## 🎯 实现步骤

1. **第一步：绘制牌桌**
   - 创建椭圆形 div
   - 添加背景、边框、阴影
   - 确保牌桌清晰可见

2. **第二步：计算座位位置**
   - 使用椭圆方程
   - 精确定位每个座位
   - 确保座位紧贴牌桌边缘

3. **第三步：添加座位内容**
   - 头像、昵称、筹码
   - 座位朝向中心
   - 空座位显示"空位"

4. **第四步：中央区域**
   - 公共牌区域
   - 底池显示
   - 阶段提示

5. **第五步：动画效果**
   - 发牌动画
   - 筹码飞向中央
   - 比牌特效
