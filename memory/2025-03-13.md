# 2025-03-13 前端工作日志

**供 Agent 回顾当日改动与原因。**

## 概述

牌桌页（templates/index.html + static/style.css + static/script.js）布局与信息展示优化：座位手牌/信息分离、行动区精简、桌面中央与底部座位防重叠、边池展示与清理逻辑。

---

## 1. 座位：手牌与用户信息分离

- **需求**：2 张手牌从用户信息框挪出，单独摆放；手牌在左或右，不要在上方。
- **实现**：
  - **HTML**：手牌 `.player-hand-wrap` 移出 `.player-info`，与 `.player-info` 一起包在 `.seat-main` 内；筹码/表情/倒计时仍在座位根下。
  - **CSS**：仅对 `.seat-main` 做 `display: flex; flex-direction: row`，手牌 `order: 0`、信息 `order: 1`，座位本身不改动，避免 `position: absolute` + `transform` 的桌面定位错乱。
  - **弃牌灰化**：手牌不在 `.player-info` 内后，改用 `.player-seat-folded .player-hand .card-back`；JS 在渲染时给座位加 `player-seat-folded` class。
- **文件**：script.js（seatDiv.innerHTML、seatCountdownHtml 前加 `</div>` 闭合 seat-main）, style.css（.seat-main 横向 flex，.player-seat-folded）

---

## 2. 行动区横条移除

- **需求**：弃牌等按钮上面的横条没必要，移除。
- **实现**：去掉 `.actions-top-row` 包裹，「开始对局」按钮直接作为 `.actions` 子元素；删除 .actions-top-row 的 CSS（背景、边框、圆角等）。
- **文件**：templates/index.html, static/style.css

---

## 3. 手牌/筹码与头像间距

- **需求**：2 张手牌和筹码更靠近头像。
- **实现**：`.seat-main` 的 `gap: 10px` → `4px`；筹码相对座位的偏移 `20px` → `6px`（后续底部座位又改为 18px 防重叠）；`.player-info` 的 `padding: 8px 12px` → `6px 8px`。
- **文件**：static/style.css

---

## 4. 桌面中央与底部玩家筹码重叠

- **现象**：本街最高下注、需跟注、边池 1–4 与最下方玩家筹码叠在一起。
- **实现**：
  - 牌桌中央上移：`.table-center` 的 `top: 50%` → `46%`。
  - 中央信息紧凑：`.pot` 的 margin/gap 缩小，`.street-bet-call`、`.side-pots` 字号与 margin 缩小。
  - 底部座位筹码上移：`data-seat-display="0","1","5","6"` 的筹码 `translate(-50%, -6px)` → `translate(-50%, -18px)`。
- **文件**：static/style.css

---

## 5. 边池一行展示与无数据时清理

- **需求**：边池写成一行；没有边池时不显示边池，并及时清理。
- **实现**：
  - **CSS**：`.side-pots` 增加 `white-space: nowrap`；`.side-pot-item` 设为 `display: inline`；`.side-pot-item + .side-pot-item::before { content: ' · ' }` 做分隔。
  - **JS**：仅根据 `state.side_pots && state.side_pots.length > 0` 判断；无边池时 `innerHTML = ''`、`display = 'none'`；有边池时渲染为一行并 `display = 'block'`。不再单独因 `stage === 'ended'` 清空。
- **文件**：static/style.css, static/script.js
