# 德州扑克 10 套风格：基于现有设计的衍生与优化

本文档在现有牌桌/大厅视觉体系基础上，抽象出「基础风格」并衍生 10 套可切换主题，便于后续前端实现主题切换（如 CSS 变量 + 类名或独立样式文件）。

---

## 一、现有基础风格摘要（Base）

当前 dzpokerV3 已形成的统一语言：

| 维度 | 现状 |
|------|------|
| **背景** | 深色房间：`#0a1219` → `#13202d` 径向渐变 + 极弱噪点纹理 |
| **牌桌** | 椭圆绿毡 `#0d4d2b` / `#0f5c34`，木质边框 `#4a3728` / `#6b4c3a`，内阴影 + 外阴影 |
| **强调色** | 琥珀金 `#f5c542`（筹码、底池、行动高亮、按钮、导航） |
| **文字** | 主色 `#e8ecf0`，次要 `#8a96a4`；数字用 JetBrains Mono |
| **字体** | Noto Sans SC（中文）、JetBrains Mono（筹码/底池） |
| **卡片** | 白底 `#fafafa`，红黑花色 `#c41e3a` / `#1a1a1a`；牌背深蓝渐变 + 金边 |
| **圆角** | 牌 12px，面板 16px，按钮 10px |
| **氛围** | 沉稳、赌场 VIP 感、低饱和度、金绿主调 |

下文 10 套风格均**保留**：椭圆牌桌、座位布局、行动按钮逻辑、发牌/下注动画思路；**可变**：配色、质感、字体、装饰程度。

## 二、10 套风格设计与优化

### 风格 1：经典绿金（当前默认 · 优化版）

- **定位**：在现有基础上微调对比度与可读性，作为默认主题。
- **调整建议**：略提高 `--text-primary` 与 `--accent-amber` 的对比度；牌桌绿毡增加一点冷暖平衡（略偏青绿），其余保持。
- **适用**：默认主题、桌面端主推。

---

### 风格 2：深蓝赌场（Blue Casino）

- **主色**：背景 `#0b1629` → `#0f2137`；牌桌毡 `#0e3d5c` / `#125a7a`（深蓝绿）；强调色 `#5eb8e8`（亮蓝）或 `#7dd3fc`。
- **辅色**：木质边框改为深灰蓝 `#2d3d4f`；筹码/底池/高亮统一用亮蓝。
- **字体**：可沿用 Noto Sans SC + Mono，或换为 Inter + JetBrains Mono 更偏数字产品。
- **氛围**：冷静、现代赌场，偏「海洋/夜空」。

---

### 风格 3：酒红丝绒（Burgundy Velvet）

- **主色**：背景 `#1a0f12` → `#2d1520`；牌桌毡 `#4a1c28` / `#6b2435`（酒红）；强调色 `#e8b4a0` 或 `#d4a574`（金棕）。
- **辅色**：木质 `#3d2820`；按钮/高亮用金棕或淡金。
- **氛围**：复古、高端会所、丝绒感。

---

### 风格 4：午夜黑金（Midnight Black & Gold）

- **主色**：背景纯黑 `#0a0a0a` → `#141414`；牌桌深灰 `#1c1c1c` / `#252525`（几乎黑毡）；强调色 `#d4af37`（古典金）。
- **辅色**：细金线边框、牌背金纹；红黑牌面保持，提高牌面与背景对比。
- **氛围**：极简、高端、强调金色点缀。

---

### 风格 5：森林绿（Forest Green）

- **主色**：背景 `#0d1410` → `#152018`；牌桌毡 `#1b4d2e` / `#246336`（森林绿）；强调色 `#86efac` 或 `#4ade80`（翠绿）。
- **辅色**：木色偏棕绿 `#3d4a2e`；按钮主色用翠绿渐变。
- **氛围**：自然、放松、偏「户外/俱乐部」。

---

### 风格 6：紫罗兰 VIP（Violet VIP）

- **主色**：背景 `#120d18` → `#1e1528`；牌桌毡 `#2d1f4e` / `#3d2a6b`（深紫）；强调色 `#c084fc` 或 `#a78bfa`。
- **辅色**：与 style_new 大厅紫卡统一；木质可改为深紫灰。
- **氛围**：神秘、VIP 包厢、偏电竞/年轻。

---

### 风格 7：暖黄灯光（Warm Lamp）

- **主色**：背景 `#1c1810` → `#2a2318`；牌桌毡 `#3d3520` / `#5c4d2a`（暖棕绿）；强调色 `#fbbf24`（暖黄）。
- **辅色**：整体偏暖；阴影略偏橙褐；字体与按钮保持高对比。
- **氛围**：老派扑克室、台灯下、怀旧。

---

### 风格 8：高对比竞技（High Contrast Arena）

- **主色**：背景 `#0f0f0f`；牌桌 `#1a1a1a`；强调色 `#22d3ee`（青）或 `#f472b6`（粉），任选其一做主色。
- **辅色**：大块纯色、少渐变；边框用高亮色细线。
- **氛围**：电竞/直播感、清晰、强对比。

---

### 风格 9：极简浅色（Light Minimal）

- **主色**：背景 `#e8ecf0` → `#f1f5f9`；牌桌毡 `#94a3b8` / `#cbd5e1`（浅灰蓝）；强调色 `#0f766e`（深青）或 `#b45309`（深琥珀）。
- **辅色**：深色文字 `#1e293b`；牌面保持红黑；阴影轻、圆角可略大。
- **氛围**：日间、清爽、低刺激，适合长时间游戏。

---

### 风格 10：赛博霓虹（Cyber Neon）

- **主色**：背景 `#0a0a12` → `#12121f`；牌桌深紫黑 `#1e1b2e`；强调色 `#00ff88`（霓虹绿）或 `#ff00aa`（品红），可双强调。
- **辅色**：发光描边、轻微 glow；字体可换为更几何的无衬线。
- **氛围**：赛博、未来、霓虹灯牌感。

## 三、实现建议（给前端）

1. **CSS 变量**：将「基础风格」中所有颜色、圆角、阴影抽成 `:root` 变量；每套主题对应一份变量覆盖（如 `[data-theme="violet"]` 或 `.theme-violet`）。
2. **牌桌与牌背**：牌桌背景、木质边框、牌背渐变建议一并纳入主题变量，避免硬编码。
3. **字体**：若某几套主题要换字体，可用 `--font-sans` / `--font-mono` 变量，默认指向当前 Noto Sans SC + JetBrains Mono。
4. **无障碍**：每套主题需保证正文与背景对比度 ≥ 4.5:1，强调按钮/筹码与背景 ≥ 3:1。
5. **命名**：建议英文 key 便于代码与文档一致，例如 `blue-casino`、`burgundy-velvet`、`midnight-black-gold`、`forest-green`、`violet-vip`、`warm-lamp`、`high-contrast-arena`、`light-minimal`、`cyber-neon`；「经典绿金」可为 `classic` 或 `default`。

## 四、小结

| # | 名称（建议 key） | 主调 | 一句话 |
|---|------------------|------|--------|
| 1 | classic / default | 绿毡 + 琥珀金 | 当前默认的微调版 |
| 2 | blue-casino | 深蓝毡 + 亮蓝 | 冷静现代赌场 |
| 3 | burgundy-velvet | 酒红毡 + 金棕 | 复古丝绒会所 |
| 4 | midnight-black-gold | 黑毡 + 古典金 | 极简黑金 |
| 5 | forest-green | 森林绿毡 + 翠绿 | 自然俱乐部 |
| 6 | violet-vip | 深紫毡 + 紫/粉 | 神秘 VIP |
| 7 | warm-lamp | 暖棕毡 + 暖黄 | 台灯怀旧 |
| 8 | high-contrast-arena | 黑灰 + 青/粉 | 电竞高对比 |
| 9 | light-minimal | 浅灰毡 + 深青/琥珀 | 日间极简 |
| 10 | cyber-neon | 紫黑 + 霓虹绿/品红 | 赛博霓虹 |

以上 10 套均在「同一套牌桌布局与交互」前提下，仅通过配色与质感变化形成不同风格，便于后续选 2～3 套先落地为可切换主题，其余作备选。
