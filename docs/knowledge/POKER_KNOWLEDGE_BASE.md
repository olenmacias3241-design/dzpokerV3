## 标准规则（实现与裁判依据）

以下为**标准德州扑克**的规则定义，比牌与底池分配逻辑必须与此一致。

### 牌型大小（Hand Rankings）

从大到小共 9 种牌型，同牌型时按「比较规则」决定胜负。

| 顺序 | 牌型（英文） | 牌型（中文） | 说明 |
|------|--------------|--------------|------|
| 1 | Royal Flush | 皇家同花顺 | 同花色的 T-J-Q-K-A |
| 2 | Straight Flush | 同花顺 | 同花色五张连续点数 |
| 3 | Four of a Kind | 四条 | 四张同点数 |
| 4 | Full House | 葫芦 | 三条 + 一对 |
| 5 | Flush | 同花 | 五张同花色，不要求连续 |
| 6 | Straight | 顺子 | 五张点数连续，花色可不同 |
| 7 | Three of a Kind | 三条 | 三张同点数 |
| 8 | Two Pair | 两对 | 两个对子 |
| 9 | One Pair | 一对 | 一个对子 |
| 10 | High Card | 高牌 | 无成牌，比单张大小 |

**点数顺序（用于比大小）：** 2 < 3 < 4 < 5 < 6 < 7 < 8 < 9 < T < J < Q < K < A。A 在顺子中可当 1 用（见下）。

### 顺子与 A-2-3-4-5（Wheel）

- **顺子**：五张牌点数连续即可，花色任意。例如 8-9-T-J-Q、A-2-3-4-5。
- **A-2-3-4-5（最小顺子 / Wheel）**：A 在此处视为 1，因此 2-3-4-5-A 是顺子，且是最小顺子；6-7-8-9-T 比它大。
- **同花顺大小**：先比顺子最高张。例如 9-T-J-Q-K 同花顺 > 8-9-T-J-Q 同花顺；A-2-3-4-5 同花顺是最小同花顺。

### 同牌型比较规则（Tie-Breaking）

- **四条 / 葫芦 / 三条 / 两对 / 一对**：先比主牌（四条比点数、葫芦比三条点数、三条比点数、两对先比大对再比小对、一对比点数），再按需要比**踢脚（Kicker）**：从大到小比单张，直到分出胜负。
- **同花 / 高牌**：从最大牌开始逐张比较（A 最大，2 最小）。
- **顺子 / 同花顺**：比顺子的**最高张**即可（5-6-7-8-9 与 6-7-8-9-T 比 9 与 T）。
- **完全相同**：则**平分底池**；无法均分的奇数筹码，按本项目约定分配给「位置最靠前（离 Dealer 最近）」的玩家。

### 一局流程（标准德州）

1. **发 2 张底牌** → **底牌圈下注（Preflop）**
2. **发 3 张公共牌（Flop）** → **翻牌圈下注**
3. **发 1 张转牌（Turn）** → **转牌圈下注**
4. **发 1 张河牌（River）** → **河牌圈下注**
5. **比牌（Showdown）** → 按牌型与边池规则分池，牌局结束。

### 座位与行动顺序

- 所有行动按**座位顺时针**（player_order / seat index 递增）。
- **位置定义**：Dealer（庄/BT）→ Small Blind（SB）→ Big Blind（BB）→ UTG → … 顺时针。
- **底牌圈（Preflop）**：  
  - 先下盲注（SB、BB 强制下注；不足则 All-in）。  
  - **第一个行动** = BB 下一位（UTG），即 `(bb_pos + 1) % n`。  
  - **最后行动** = BB（无人加注时 BB 可 Check）。
- **翻牌圈 / 转牌圈 / 河牌圈（Flop / Turn / River）**：  
  - **第一个行动** = 庄位（BT）下一位起，第一个**未弃牌且未 All-in** 的玩家；即从 `(dealer_pos + 1) % n` 起顺时针找。

### 下注轮结束条件

- 从「当前行动者」顺时针找下一个「未弃牌、未 All-in、且本街投入 < 当前跟注额」的玩家；若找到则轮到他行动。
- 若绕一圈回到**最后加注者**，且所有人都已跟注到同一金额，则本街结束，发下一张公共牌或进入摊牌。

### 底池与边池（Side Pots）

- **算法须支持多个边池。**
- 流程概要：  
  1. 将所有 All-in 玩家按 All-in 金额**从小到大**排序。  
  2. 最小的 All-in 金额定义**主池（Main Pot）**；计算所有玩家投入该池的筹码。  
  3. 后续 All-in 金额依次定义 Side Pot 1、Side Pot 2…  
  4. 从**最后一个边池**开始，在有资格赢取该池的玩家中取牌型最大者分配；再向前处理，直到主池分配完毕。  
  5. 平分时，奇数筹码分配给位置最靠前（离 Dealer 最近）的玩家。

## 基本术语

_以下术语与策略部分供产品文案、AI 对话、策略参考使用，不作为裁判规则。_

### 游戏阶段 (Game Stages)
- **翻牌前 (Pre-flop)**：发底牌后、公共牌发出前的阶段
- **翻牌圈 (Flop)**：发出前三张公共牌后的阶段
- **转牌圈 (Turn / Fourth Street)**：发出第四张公共牌后的阶段
- **河牌圈 (River / Fifth Street)**：发出第五张（最后一张）公共牌后的阶段
- **摊牌 (Showdown)**：比较牌型大小，决定胜负

### 玩家行动 (Player Actions)
- **下注 (Bet)**：当无人下注时，主动投入筹码
- **跟注 (Call)**：投入与前面玩家相同数量的筹码
- **加注 (Raise)**：增加当前下注额
- **再加注 (Re-raise)**：对已有的加注再次进行加注
- **过牌 (Check)**：不投入筹码，将行动权传给下一位
- **弃牌 (Fold)**：放弃当前手牌，退出本局
- **全下 (All-In / Shove / Jam / Push)**：投入所有剩余筹码

### 高级行动术语
- **持续下注 (Continuation Bet / C-Bet)**：翻牌前加注者在翻牌后继续下注
- **过牌-加注 (Check-Raise)**：先过牌，等对手下注后再加注反击
- **溜入 (Limp)**：仅跟注大盲注进入翻牌圈（通常不推荐）
- **3-Bet**：对一个加注进行再加注（第三次下注）
- **4-Bet**：对一个3-Bet进行再加注（第四次下注）
- **价值下注 (Value Bet)**：用强牌下注，期望更弱的牌跟注
- **诈唬 (Bluff)**：用弱牌下注或加注，试图迫使强牌弃牌
- **半诈唬 (Semi-Bluff)**：带有听牌潜力的下注
- **慢玩 (Slowplay)**：用强牌被动打（过牌/跟注）以诱导对手继续投入筹码
- **浮打 (Float)**：在位置上用较弱的牌跟注，计划在后续街拿下底池
- **超额下注 (Overbet)**：下注金额大于底池大小
- **堵塞下注 (Block Bet)**：小额下注以控制底池大小或阻止对手进行大额下注
- **冷跟 (Cold Call)**：本轮尚未投入筹码的玩家直接跟注加注

### 位置术语 (Position Terms)
- **按钮位/庄家位 (Button / BTN)**：各轮下注中最后行动的位置，最有优势
- **小盲注 (Small Blind / SB)**：按钮位左侧第一个玩家，下强制性小额盲注
- **大盲注 (Big Blind / BB)**：小盲注左侧第一个玩家，下强制性大额盲注
- **枪口位 (Under the Gun / UTG)**：大盲注左侧第一个行动的玩家，翻牌前第一个行动
- **UTG+1 / UTG+2**：紧随UTG之后的早期位置
- **低位劫持 (Lowjack / LJ)**：中等位置
- **劫持位 (Hijack / HJ)**：中后位，在Cutoff之前一位
- **关煞位 (Cutoff / CO)**：按钮位右侧一位，第二强的偷盲位置
- **早期位置 (Early Position / EP)**：UTG、UTG+1等，需要较紧的起手牌范围
- **中间位置 (Middle Position / MP)**：Lowjack、Hijack等，范围灵活度中等
- **后期位置 (Late Position / LP)**：Cutoff、Button等，行动靠后，优势较大

### 手牌与牌力术语
- **坚果牌 (Nuts)**：在当前牌局中能组成的最大可能牌型
- **踢脚牌 (Kicker)**：组成牌型后，用于比较牌大小的单张牌
- **连牌 (Connectors)**：点数相连的底牌，如6-7
- **口袋对子/底对 (Pocket Pair)**：两张底牌组成一对
- **同花牌 (Suited)**：两张底牌花色相同
- **杂色牌 (Off Suit)**：两张底牌花色不同
- **大牌/高牌 (Over Card)**：比公共牌中任何一张牌都大的底牌
- **听牌 (Draw Hand / Drawing Hand)**：尚未组成成牌，但有机会在后续公共牌发出后组成
- **中洞顺子听牌 (Gutshot Straight Draw)**：只差一张牌就能组成顺子，且这张牌在顺子的中间
- **两头顺子听牌 (Open-Ended Straight Draw)**：只差一张牌就能组成顺子，且这张牌在顺子的两端
- **后门听牌 (Backdoor / Runner)**：需要转牌和河牌连续发出特定牌才能组成顺子或同花
- **三条 (Trips)**：用一张底牌和两张公共牌组成的三条
- **顶对顶踢脚 (Top Pair Top Kicker / TPTK)**：拥有最大的对子和最大的踢脚牌
- **无关牌 (Blank)**：发出后不影响当前牌局强度或胜负的公共牌
- **底对 (Bottom Pair)**：用底牌与最低的公共牌组成的一对
- **死亡听牌 (Drawing Dead)**：即使后续发出任何牌也无法获胜的听牌状态
- **超对 (Overpair)**：底牌对子大于公共牌中任何一张牌
- **彩虹面 (Rainbow)**：公共牌花色各不相同，无法组成同花

### 玩家类型
- **紧手玩家 (Nit)**：玩的非常紧，几乎只玩最好的起手牌
- **磨蹭家 (Grinder)**：通过小心谨慎、一点点积累筹码来取得成绩
- **鱼 (Fish)**：指牌技较差，容易输掉筹码的玩家
- **马脚 (Tell)**：玩家不经意间暴露牌力强弱的表情、动作或言语
- **慢亮牌 (Slow Roll)**：明明赢牌却故意拖延亮牌，被视为非常差的牌桌礼仪

### 其他常见术语
- **底池 (Pot)**：每手牌所有玩家投入的筹码总额
- **盲注 (Blinds)**：每局开始前由小盲位和大盲位玩家支付的强制性下注
- **前注 (Ante)**：锦标赛后期或某些现金局中，每位玩家在发牌前必须支付的小额强制下注
- **抽水 (Rake)**：赌场或线上扑克室从每局底池中按比例抽取的费用
- **资金 (Bankroll)**：玩家用于进行扑克游戏的资金总量
- **两人对决/单挑 (Heads Up)**：牌桌上只剩下两位玩家进行对决
- **爆冷门 (Bad Beat)**：原本胜算很高的牌意外被击败
- **优势 (Edge)**：玩家相对于对手的技能、位置、筹码等方面的有利条件
- **碎牌 (Cracked)**：当一对漂亮的底牌（如AA）被对手击败
- **Tilt（上头）**：情绪失控导致打法变差

## 游戏流程（与规则一致）

与「标准规则」一致，便于实现与裁判对照。

### 标准流程
1. **发底牌**：每位玩家发两张底牌（Hole Cards）
2. **翻牌前下注（Preflop）**：从 **UTG（BB 下一位）** 开始，顺时针行动；**BB 最后行动**（无人加注时 BB 可 Check）
3. **发翻牌（Flop）**：发出三张公共牌
4. **翻牌圈下注**：从 **庄位（BT）下一位**起第一个未弃牌且未 All-in 的玩家开始，顺时针行动
5. **发转牌（Turn）**：发出第四张公共牌
6. **转牌圈下注**：同上，从 BT 下一位起
7. **发河牌（River）**：发出第五张公共牌
8. **河牌圈下注**：同上，从 BT 下一位起
9. **摊牌（Showdown）**：按牌型与边池规则比较，最大牌型赢得对应底池

### 盲注机制
- **小盲注（SB）**：通常是大盲注的一半
- **大盲注（BB）**：基础下注单位
- **盲注轮换**：每手牌结束后，庄家按钮顺时针移动一个位置
- **盲注增长**：锦标赛中盲注会随时间逐渐增加

## 位置与策略

### 位置的重要性
**位置被认为是决定一切的关键因素。** 玩家的行动顺序会随庄家按钮顺时针移动而轮换。

### 位置分类与优劣

#### 早期位置 (Early Position / EP)
- **位置**：UTG、UTG+1、UTG+2
- **特点**：需率先行动，信息最少
- **策略**：采取更紧（conservative）的策略，只玩强牌范围
- **起手牌范围**：AA, KK, QQ, JJ, AKs, AQs 等顶级牌

#### 中间位置 (Middle Position / MP)
- **位置**：Lojack (LJ)、Hijack (HJ)
- **特点**：比早期位置有更多信息
- **策略**：可适度放宽起手牌范围，但仍需保持警惕
- **起手牌范围**：可加入 TT, 99, AJs, KQs 等

#### 晚期位置 (Late Position / LP)
- **位置**：Cutoff (CO)、Button (BTN)
- **特点**：拥有最大的优势
- **优势**：
  - **信息优势**：可以看到前面玩家的行动
  - **控制底池**：更好地控制底池大小
  - **策略灵活性**：能根据对手反应调整策略
- **起手牌范围**：可以玩更宽的范围，包括同花连张、小对子等投机牌

#### 盲注位置 (Blinds / SB & BB)
- **特点**：
  - 翻牌前是强制下注者
  - 翻牌后通常需要率先行动（SB最先，若SB弃牌则BB先）
  - 成为最难打的位置
- **大盲注防守**：由于已投入筹码，面对加注时可以更宽的范围防守
- **小盲注策略**：在小盲位开池时可以放宽范围，因为对手只剩一名

### 位置与起手牌范围
**核心原则：位置越靠后，可以玩的起手牌范围越宽。**

## 数学基础

### 1. 底池赔率 (Pot Odds)

底池赔率是衡量你是否应该跟注的重要工具。

**计算公式：**
```
底池赔率 = 跟注所需筹码 / 底池大小
或
底池赔率 = (底池总额 ÷ 跟注需要的金额) : 1
```

**转换为所需胜率百分比：**
```
所需胜率 = 1 / (1 + 底池赔率的比例部分)
```

**计算示例：**
- 底池：12,000筹码
- 对手全下：6,000筹码
- 你需要跟注：6,000筹码

1. 跟注后底池总额：12,000 + 6,000 = 18,000筹码
2. 底池赔率：(18,000 ÷ 6,000) : 1 = 3 : 1
3. 所需胜率：1 / (1 + 3) = 25%

**结论：** 你至少需要有25%的赢牌几率才能使跟注在长期来看是收支平衡或盈利的。

### 2. 隐含赔率 (Implied Odds)

隐含赔率是底池赔率的延伸，它不仅考虑当前的底池大小，还把你在后续下注回合中预期能从对手那里赢取的额外筹码也考虑在内。

**核心思想：**
在计算底池赔率时，将"底池总额"部分加上你预期在未来回合中可能从对手那里赢取的额外筹码。

**计算示例：**
- 转牌圈，你听同花
- 当前底池赔率需要25%胜率
- 你的实际胜率只有18%
- 预计河牌成功后对手还会投入60筹码

```
隐含赔率所需胜率 = 投入筹码 / (投入后底池 + 预期额外收益)
例如：20 / (80.5 + 60) = 20 / 140 ≈ 14%
```

由于你的实际胜率18% > 14%，因此跟注是+EV的。

**反向隐含赔率 (Reverse Implied Odds)：**
即使你击中了所需的牌，最终仍可能输掉一个大底池的情况。

### 3. 期望值 (Expected Value - EV)

期望值代表了你在长期游戏中，每次做出特定决策后平均预期会赢得或损失多少筹码。

**计算公式：**
```
EV = (赢率 × 潜在收益) - (输率 × 潜在损失)
或
EV = (获胜概率 × 赢取的金额) – (失败概率 × 损失的金额)
```

**计算示例：**
- 获胜概率：40%
- 获胜收益：100筹码
- 失败概率：60%
- 失败损失：50筹码

```
EV = (0.40 × 100) - (0.60 × 50)
EV = 40 - 30
EV = +10筹码
```

**结论：** 这是一个正期望值（+EV）的决策，长期来看每次会盈利10筹码。

### 4. 出牌数 (Outs)

出牌数是指能让你的牌改进为赢牌的剩余牌数。

**常见听牌的出牌数：**
- **同花听牌**：9张（13张同花色 - 4张已知）
- **两头顺子听牌**：8张（两端各4张）
- **中洞顺子听牌**：4张
- **对子听三条**：2张
- **两对听葫芦**：4张

**快速计算胜率（Rule of 2 and 4）：**
- **转牌前**：出牌数 × 4 ≈ 胜率%
- **河牌前**：出牌数 × 2 ≈ 胜率%

**示例：**
- 翻牌圈有9个同花听牌
- 转牌+河牌胜率：9 × 4 = 36%
- 仅河牌胜率：9 × 2 = 18%

## 高级策略

### 1. GTO (Game Theory Optimal) 策略

**定义：** GTO是一种理论上最平衡、无法被对手剥削的防御性策略。它旨在达到纳什均衡（Nash Equilibrium）。

**核心特点：**
- **不可被剥削**：任何一方都无法通过单方面改变策略来获得优势
- **防御性**：主要目标是"不被剥削"，而非"打爆"对手
- **平衡性**：让对手无法通过你的行为模式判断你的手牌强度

**GTO的应用：**
- 构建精确且平衡的翻牌前和翻牌后手牌范围
- 在下注范围中同时包含强价值牌和诈唬牌
- 以一定的比例混合，让对手难以区分

### 2. 范围 (Range) 思维

**定义：** 范围是指在特定情况下，对手（或你）可能持有的所有手牌的集合。

**核心原则：**
- **不要猜单一手牌**：优秀的玩家不会只猜测对手的某一张牌
- **推断范围**：推断对手可能持有的所有手牌组合范围
- **范围vs范围**：用范围而非单一手牌来思考

**范围的应用：**
- 根据对手的位置、行动推断其范围
- 根据公共牌面调整对手的范围
- 根据范围做出最优决策

### 3. 平衡 (Balance)

**定义：** 平衡是GTO策略的核心思想，指的是在你的打法中，要让对手无法通过你的行为模式来准确判断你的手牌强度。

**平衡的表现：**
- **下注范围平衡**：同时包含强价值牌和诈唬牌
- **行动频率平衡**：不同行动（下注/过牌/加注）的频率合理
- **不可被读**：对手难以区分你是在诈唬还是真的有强牌

**平衡的好处：**
- 避免被对手轻易剥削
- 让对手的决策变得困难
- 长期来看更稳定

### 4. 剥削 (Exploitation)

**定义：** 剥削是一种进攻型策略，旨在识别并利用对手的弱点或非最优打法来最大化自己的收益。

**剥削的应用：**
- **识别对手弱点**：过度弃牌、过度跟注、过度诈唬等
- **针对性调整**：故意偏离GTO策略，采取更具针对性的打法
- **最大化收益**：惩罚对手的系统性错误

**剥削的风险：**
- **可被反剥削**：你故意改变频率的同时，也可能使自己变得"可被反剥削"
- **需要准确读牌**：如果读错对手，可能适得其反

**GTO vs 剥削：**
- **GTO是基础**：确保你"不输"
- **剥削是加速器**：帮助你"赚更多"
- **顶尖玩家**：在理解GTO的基础上，根据对手的特定行为偏差灵活地切换到剥削策略

### 5. 持续下注 (C-Bet) 策略

**定义：** 在翻牌前加注的玩家在翻牌后继续下注。

**C-Bet的目的：**
- **维持主动性**：保持翻牌前的进攻态势
- **赢取底池**：迫使对手弃牌
- **建立形象**：让对手难以判断你的牌力

**C-Bet的频率：**
- **不是100%**：需要根据牌面、对手、位置调整
- **平衡范围**：既有价值牌也有诈唬牌
- **牌面考量**：干燥牌面（如K-7-2彩虹）更适合C-Bet

### 6. 控池 (Pot Control)

**定义：** 当牌力中等或牌面不利时，通过过牌来控制底池大小。

**控池的时机：**
- **中等牌力**：如顶对中等踢脚
- **牌面危险**：如三张同花、三张连牌
- **对手激进**：对手可能有更强的牌

**控池的好处：**
- **降低风险**：避免投入过多筹码
- **保留选择权**：后续可以根据情况调整
- **诱导诈唬**：给对手诈唬的机会

## 锦标赛策略

### 1. ICM (Independent Chip Model)

**定义：** ICM是一种数学模型，用于在锦标赛中将筹码价值转换为真实货币价值（锦标赛权益）。

**核心概念：**
- **筹码价值非线性**：在锦标赛中，筹码价值不是线性的
- **生存价值**：避免被淘汰比赢取筹码更重要
- **奖金跳跃**：接近奖金圈或决赛桌时，奖金跳跃显著

### 2. ICM 压力 (ICM Pressure)

**定义：** ICM压力是指在锦标赛中，特别是在奖金圈（Bubble）和决赛桌附近，由于奖金跳跃而产生的财务压力。

**泡沫因子 (Bubble Factor)：**
- **高泡沫因子**：需要打得更紧，因为输掉筹码比赢得筹码更有害
- **低泡沫因子**：可以打得更松

### 3. 基于筹码量的策略调整

#### 大筹码 (Big Stack)
- **ICM压力最小**：不太可能被淘汰
- **策略**：
  - 对中短筹码施加最大压力
  - 扩大开牌范围、3-Bet范围、Shove范围
  - 攻击盲注
  - **避免与其他大筹码冲突**

#### 中筹码 (Medium Stack)
- **ICM棺材**：面临来自大筹码的压力，又想超越短筹码
- **策略**：
  - 更加风险规避
  - 专注于"爬梯子"（Laddering Up）
  - 避免边缘情况
  - 打小底池
  - **弃牌可以是盈利的决策**

#### 短筹码 (Short Stack)
- **ICM压力最大**
- **策略**：
  - 通常需要打得更紧
  - 寻找机会翻倍，特别是针对其他短筹码
  - **极端情况**：在卫星赛中，ICM压力可能极端到需要弃掉AA

### 4. 常见ICM错误

- **跟注范围过宽**：没有考虑ICM压力
- **忽略相对筹码量**：没有考虑对手的筹码量
- **忽略奖金跳跃**：没有意识到接近奖金圈
- **大筹码过于被动**：没有充分利用筹码优势
- **高估手牌强度**：把ICM当作二元决策（要么超紧要么超松）

### 5. 卫星赛 (Satellites) 的极端ICM

**特点：**
- **奖励是席位**：而非奖金分配
- **额外筹码无价值**：超过席位所需的筹码没有额外价值
- **生存至上**：避免被淘汰是唯一目标

**策略：**
- **极度保守**：即使拿到强牌也可能需要弃牌
- **避免风险**：取得席位比赢取更多筹码重要
- **计算席位线**：知道需要多少筹码才能获得席位

## 实战应用建议

### 1. 学习路径
1. **掌握基础**：术语、规则、牌型
2. **理解数学**：底池赔率、期望值、出牌数
3. **学习位置**：不同位置的起手牌范围
4. **研究GTO**：理解平衡策略的基础
5. **练习剥削**：识别对手弱点并调整策略
6. **锦标赛ICM**：理解筹码价值的非线性

### 2. 常见错误避免
- **不要玩太多手牌**：耐心是关键
- **不要忽略位置**：位置决定一切
- **不要情绪化**：避免Tilt
- **不要忽略数学**：底池赔率和期望值是基础
- **不要过度诈唬**：平衡是关键
- **不要忽略对手**：观察和调整

### 3. 持续改进
- **复盘**：回顾自己的牌局，找出错误
- **学习**：观看教学视频、阅读书籍
- **讨论**：与其他玩家交流策略
- **使用工具**：GTO求解器、HUD统计
- **记录**：记录自己的打法和结果
