# 下注轮次说明（Turn Order）

本文档用一句话说明「轮到谁」和「何时结束」，与 `core/game_logic.py` 实现一致。

---

## 1. 顺序约定

- **行动顺序** = **座位顺序**（顺时针，即 `player_order` / seat 0 → 1 → 2 → …）。
- **庄位 (BT)**、**小盲 (SB)**、**大盲 (BB)** 由 `dealer_button_position` 推导：
  - SB = (dealer_pos + 1) % n
  - BB = (dealer_pos + 2) % n

---

## 2. 谁先行动

| 阶段 | 第一个行动的人 |
|------|----------------|
| **底牌圈 (Preflop)** | BB 下家，即 UTG = (bb_pos + 1) % n |
| **翻牌/转牌/河牌** | 庄位下家，即 (dealer_pos + 1) % n 起第一个「未弃牌且未 All-in」的玩家 |

---

## 3. 下一个轮到谁

- 从**当前行动者**的下一位开始，按**座位顺序**顺时针找：
  - **未弃牌** 且 **未 All-in** 且 **本街投入 < 当前跟注额** → 轮到他行动。
- 若绕一圈回到**最后加注者**，且所有人都已跟齐 → **本街结束**，发下一张牌或比牌。

---

## 4. 状态字段

- `current_player_id`：当前必须行动的一名玩家；`None` 表示本街无人需行动（即将发牌或比牌）。
- `last_raiser_id`：本街最后加注的玩家，用于判断「行动是否回到最后加注者」。

---

## 5. 示例（2 人）

- 座位 0、1 有人，dealer_pos = 0 → SB=1，BB=0。
- 底牌圈先行动：座位 1（SB）；后行动：座位 0（BB）。
- 翻牌/转/河：庄位下家 = 座位 1 先行动。

---

## 6. 示例（3 人）

- 座位 0、1、2，dealer_pos = 0 → SB=1，BB=2。
- 底牌圈行动顺序：0(UTG) → 1(SB) → 2(BB)。
- 翻牌/转/河：庄位下家 = 座位 1 先行动（若未弃牌）。
