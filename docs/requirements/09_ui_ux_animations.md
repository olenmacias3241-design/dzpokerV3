# 09 - UI/UX 核心动画与道具设计方案 V1.0

本文档为动效设计师和前端开发的具体实现指南。

---

### 1. 打牌桌面 (The Playing Table)

*   **主题风格:** 现代极简主义融合了真实质感。我们称之为 **“数字丝绒 (Digital Velvet)”**。
*   **桌面背景:** 采用深邃的藏蓝色 (`#1A2C3D`)，但不是纯色。背景会有一层非常细腻、几乎看不见的**丝绒/毛毡材质的纹理 (noise texture)**，并在桌子边缘有轻微的、柔和的暗角，营造出高级感和深度。
*   **光照:** 仿佛有一盏柔和的顶灯(spotlight)从斜上方照射下来。桌子中央最亮，边缘稍暗。当发公共牌或底池筹码变动时，光照区域会有一个非常短暂、轻微的亮度提升，引导玩家的视觉焦点。
*   **牌桌线条:** 用**微光的霓虹线条** (`#FFBF57` 琥珀金) 勾勒出座位区、公共牌区和底池区的边界。这些线条在大部分时间是静态的，但在轮到某个玩家行动时，他座位区的线条会开始缓慢地“呼吸”闪烁。

### 2. 用户下注动画 (Betting Animation)

这是一个需要**极度清晰且有物理感**的动画，分为三个阶段：

1.  **意图阶段 (Intention):**
    *   玩家点击“下注/加注”按钮。
    *   他座位前的区域会微微亮起，**下注滑块和快捷按钮**以一个快速、平滑的缓动动画 (ease-out) 出现。
    *   当玩家拖动滑块时，他面前会**预显示**一堆半透明的“幽灵筹码”，数量会随着滑块的拖动实时增减，让玩家对下注量有直观的感受。

2.  **执行阶段 (Execution):**
    *   玩家确认下注。
    *   对应数量的筹码，会从玩家的总筹码堆里**分离出来**，并发出清脆的“咔嗒”声。
    *   这些筹码不会瞬间移动，而是会沿着一条**优雅的贝塞尔曲线 (bezier curve)**，以一个带有加速度的物理轨迹，快速地滑入中央底池。
    *   **关键细节:** 不是一整坨筹码滑过去，而是多枚筹码，它们在滑动过程中会有些许随机的旋转和轻微的碰撞，增加真实感。

3.  **结算阶段 (Resolution):**
    *   筹码汇入底池，并与底池中的其他筹码合并，发出“哗啦”的碰撞音效。
    *   底池总额的数字，会像老虎机数字一样，快速向上滚动，最终停在新的总额上。

### 3. 比牌动画 (Showdown Animation)

这是游戏中最激动人心的时刻，动画必须充满**仪式感和戏剧性**。

1.  **准备阶段 (The Setup):**
    *   所有未弃牌的玩家，他们的底牌会自动翻开。
    *   系统会自动识别每个玩家能组成的最大牌型。

2.  **高亮阶段 (The Highlight):**
    *   系统会用一道**流光效果**，依次划过组成最大牌型的那几张牌（2张底牌 + 3-5张公共牌）。例如，如果玩家是葫芦 (Full House)，流光会依次扫过他的对子和三条。
    *   被流光扫过的牌，外框会持续散发着与牌型等级对应的辉光（例如：同花是蓝色，葫芦是紫色，皇家同花顺是金色）。
    *   同时，**仅在自己座位（本玩家控制台）**显示清晰的、带动画效果的**牌型徽章/牌型名称** (e.g., "葫芦", "FULL HOUSE")；其他玩家的座位上**不展示牌型文字**，仅展示其底牌与高亮组成牌型的牌张（若需对比），以保证牌型信息只出现在己方控制台。

3.  **胜利阶段 (The Win):**
    *   **对比:** 如果有多个玩家，牌型最大的那个玩家的徽章和卡牌辉光会“脉冲”一下，变得更亮，其他玩家的则会黯淡下去。
    *   **收钱:** 中央底池的所有筹码，会像被一股引力吸引一样，汇聚成一股“筹码洪流”，沿着一条华丽的曲线飞向胜利者的座位，并伴有金币和粒子飞散的庆祝特效。
    *   胜利者的屏幕中央会出现一个巨大的祝贺信息，例如 **“你赢得了 25,000！”**。

### 4. 用户道具 (Player Items & Props)

这是增强社交和个性化的关键。道具分为两类：

1.  **互动消耗品 (Interactables):**
    *   **用途:** 玩家可以在牌桌上对其他玩家使用的表情道具，有冷却时间。
    *   **清单:**
        *   **扔番茄 🍅:** 对一个思考过久的玩家使用，表达催促。
        *   **献玫瑰 🌹:** 对打出漂亮操作的对手使用，表达赞赏。
        *   **丢炸弹 💣:** 当自己 All-in 时使用，增加气势。
        *   **竖白旗 🏳️:** 当自己弃掉一手大牌时使用，表达无奈。
    *   **动画:** 这些道具会以抛物线的方式从你的座位飞到目标玩家的座位，并产生一个有趣的粒子效果。

2.  **个性化装饰品 (Cosmetics):**
    *   **用途:** 长期展示，彰显玩家身份和成就。
    *   **清单:**
        *   **牌桌宠物/吉祥物:** 一个小小的、可爱的3D模型（比如一只招财猫、一条小龙），趴在你座位旁边，偶尔会有一些小动作。
        *   **自定义卡背:** 可以通过成就或购买解锁的、具有独特设计的卡牌背面。**牌面/牌背背景**：卡牌背景须为**花纹色**（图案/纹理），禁止使用纯色底；花纹需清晰可辨、不干扰牌面点数与花色的识别。
        *   **底牌特效:** 当你拿到特定的大牌（如 AA, KK, QQ）时，你的两张底牌在发下来的时候会有一个专属的、短暂的流光或火焰特效（仅自己可见）。
